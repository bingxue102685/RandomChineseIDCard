# RandomChineseIDCard
Randomly generate Chinese ID number

## 1. 随机生成20-50岁年龄的身份证号
可通过防伪验证！
## 2. 随机生成名字
1. 姓是根据2017年人口调查的姓名占比，进行随机的，随机出的姓很"真实"
2. 名是分男女，分别取常见的名，也很"真实"
